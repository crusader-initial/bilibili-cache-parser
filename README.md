# bilibili-cache-parser使用说明
1. 下载 blibili_cache_parser_win.zip
2. 解压后创建如下目录结构：
blibili_cache_parser/
├── process_videos.exe
├── ffmpeg/
    └── ffmpeg.exe
|-── video/
    └── ... (B站缓存文件)
3. 将B站缓存文件放入video文件夹
4. 运行process_videos.exe
